Miscellaneous Preferences Page

These preferences affect some of Balthisar Tidy’s miscellaneous behaviors.

The Input-Encoding Helper can be set to appear when Balthisar Tidy thinks that it didn’t open a file properly.

The New User Helper points out some of the features of Balthisar Tidy the first time Tidy is run. If you’d like to see this feature again, it can be reset here.

By choosing to show alternating row colors you have control over whether Balthisar Tidy highlights every alternating row color in the list of Tidy options. This gives you control over the legibility and appearance that you’re most comfortable working with.

Hovering behavior gives you the flexibility to decide whether to see the Tidy options as nice clean text without row upon row of controls, or to see the options in the traditional format with the option selector (the pop up list or stepper control) always present.
